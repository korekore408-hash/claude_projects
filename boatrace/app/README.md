# 競艇予想アプリ（サンプル）

単一HTMLファイルで動く、競艇（ボートレース）の予想・舟券シミュレーションのサンプルアプリです。
ビルド不要・依存ライブラリなしで、`index.html` をブラウザで開くだけで動きます。

## 使い方

```
# ブラウザで開くだけ
open boatrace/app/index.html      # macOS
# または index.html をダブルクリック
```

## 機能

- **開催場一覧** … 桐生(G3)/住之江(SG)/福岡(一般) のサンプル3場
- **出走表** … 6艇の選手情報（級別・全国/当地勝率・モーター2連率・平均ST）と勝率予測バー
- **AI予想（簡易モデル）** … 各艇の強さをスコアリングし、3連単のおすすめ買い目を上位5点表示
- **舟券シミュレーター** … 買い目と金額を選んで「レース実行」。勝率分布から着順を抽選し、的中なら想定オッズで払戻計算

## 予想ロジックの概要

各艇の「強さ」を以下の重み付き合計で算出し、ソフトマックスで勝率に変換しています。

```
強さ = 全国勝率×0.45 + 当地勝率×0.25 + モーター2連率×0.04
     + 級別補正 + ST補正 + コース有利補正
```

- **コース有利補正** … 1コースが圧倒的に有利という競艇の特性を反映（イン逃げ）
- **想定オッズ** … 控除率25%を考慮した理論値（`0.75 / 確率`）

## 注意

- 出走表・オッズはすべて**ダミーデータ**です。実運用では公式の出走表・オッズデータに差し替えてください。
- 予想は学習用の簡易モデルであり、的中を保証するものではありません。
- 舟券の購入は自己責任で。**20歳未満は舟券を購入できません。**

## 既存の分析パイプラインとの関係

リポジトリ直下の `boatrace/*.py` は、公式データのダウンロード〜特徴量生成〜機械学習予測を行う本格的なパイプラインです。
本サンプルアプリ（`boatrace/app/`）は、それとは独立した「フロントエンドのデモ」として作成しています。
将来的には、`predict_win.csv` などの出力をこのアプリに読み込ませて連携することも可能です。
